Courses that we offer

The Basics Course

This course will teach you basics of programming and things related to how the internet works. It will help you understand how the web works which will provide a solid foundation for further improving your skills.

This course is aimed to help everyone who struggles with or doesn't have any experience coding.

Front End Fundamentals

In this course you will learn the basic technologies that are used in developing Front End Applications. You will learn languages such as : HTML, CSS, JavaScript and more.

Note: This course heavily relies on knowledge that's covered in The Basics course.
Also note: Content of this course is identical to what is included in Full Stack Fundamentals course.

Back End Fundamentals

This course teaches you the basics of Back End development. Here we are covering topics such as as: PHP, working with MySQL databases and more.

Note: This course heavily relies on knowledge covered in The Basics course and some of the knowledge from Front End Fundamentals course.
Also note: Content of this course is identical to what is included in Full Stack Fundamentals course.

Full Stack Fundamentals

This course teaches you both Front End and Back End technologies. You will learn languages such as HTML, CSS, JavaScript, PHP, working with MySQL database and more.
